• TABLE: Table é a tag para criar uma tabela na página.

  • TH: th representa cada cabeçalho da tabela.

  • TR: Tr ou Table Row representa cada linha da tabela.

  • TD: Td ou Table Data representa cada célula da tabela.

  • THEAD: thead representa o cabeçado da tabela.

  • TBODY: tbody representa o corpo da tabela.

  • TFOOT: tfoot representa o rodapé da tabela.

  • ALIGN: Este atributo se usa para centralizar um elemento

  • BORDER: border define as bordas da tabela

  • CAPTION: caption define um título para a tabela

  • ROWSPAN: Rowspan é um atributo que permite a célula ocupar o número de linhas que você definir no atributo(mesclagem).

  • COLSPAN: Colspan é um atributo que permite a célula ocupar o número de colunas que você definir no atributo(mesclagem).

  • WIDTH: Um atributo que define largura.
